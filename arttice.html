Arttice is a map-centered web platform for networking and mapping cultural players and artists locally and internationally.

Working closely with Ideas Block, developers, and and a visual designer, I designed the site's architecture and user flows, and outlined the core UI patterns. This is a overview of Arttice's core features.

The need for (digital) artist-run networks

Artists, musicians, and cultural producers have always connected through collaboration and cultural events. However, current digital platforms fail to serve their needs.

In response, Arttice aims to provide an artist-run online catalogue for cultural players in Lithuania and internationally to discover each other online and through local events.

The structure of Arttice




Profiles, pages, and events

To enable a seemless transition from platforms such as Facebook, Arttice is built around an account that can in turn manage multiple pages and publish multiple events on each of them.

This enables artists to create separate pages for each practice they partake in, while also enabling cultural managers to run multiple pages at once.




UI patterns




Interface design

A map-centered search interface

Both site visitors and registered users can discover pages or events by either searching by keyword, filtering by different categories, or through navigating the map UI.




Create a page

Registration is invite-only. Both moderators and existing users can generate invites. Once a person registers, they are encouraged to create a page from their profile




The registration flow on larger devices is shorter, as each screen includes two steps instead of one.




Create an event

Once they have a page, a user can create an event from it, much like on Facebook.
